Boilerplate to create a project

New=>Project
Select Visual C#.Cross-Platform
Set project name 
Naviagate to proper enclosing folder (Location)
Click OK

Select .Net Library
Verify Xamarin.Forms is selected
Click OK

Use default Windows range.
Delete MainPage from PCL project
Add Xaml or C# New Item (XxxxPage)
Remove from XxxxPage:

    [XamlCompilation(XamlCompilationOptions.Compile)]

Copy XxxxPage class name to clipboard
Replace MainPage in App.Xaml.cs with class name
